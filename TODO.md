# Elements:  
## `ǃ` Uppercase/Increment  
Either uppercases a string or adds one to a number.  
- Arity: 1

```
Overloads:
    int a: a+1
    str a: a.upper()
```

## `¡` Lowercase/Decrement
Either lowercases a string or subtracts one from a number.
- Arity: 1

```
Overloads:
    int a: a-1
    str a: a.lower()
```

## `□` Swap case/Negate
Either swaps the case of a string or negates a number.  
- Arity: 1 

```
Overloads:
    int a: -a
    str a: a.swapcase()
```
